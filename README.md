LiquidParts
===========

LiquidParts - Application for edit system settings in liquid, be added as system app
Developer - RomzesRover, based on liquid parts by Thepasto

PagerSliding it's a library to make sliding pager like in new google play, 
you can read about this here: https://github.com/astuetz/PagerSlidingTabStrip

Supported version 4.0.3 + but it possible to port to an old version like 2.3
